# credit_calculator
Калькулятор для расчета аннуитетных платежей и суммы переплаты по кредиту

Адаптивная верстка:
Использование clamp() для плавного изменения размеров шрифтов
Перестроение колонок на мобильных устройствах
Уменьшение отступов и padding для экономии места

Улучшенные элементы управления:
Слайдеры и поля ввода перестраиваются вертикально на маленьких экранах
Кнопки увеличены для удобства касания
Улучшенная навигация по таблице платежей

Адаптивная таблица:
Горизонтальная прокрутка с инерцией на iOS
Уменьшенный размер шрифта для мобильных
Адаптивное количество строк на странице (6 на маленьких экранах, 8 на средних, 12 на десктопах)
Улучшения для касаний:
Увеличенные области касания для кнопок
Оптимизация для iOS (отключение масштабирования)
Поддержка жестов для пагинации

Адаптивные графики:
Диаграмма автоматически подстраивается под размер экрана
Легенда и подписи оптимизированы для мобильных
Оптимизация для разных ориентаций:
Улучшенное отображение в landscape режиме
Перерасчет при повороте экрана

Визуальные подсказки:
Добавлены подсказки для мобильных пользователей
Четкая индикация текущей страницы таблицы
